# MysteryMind 🕵️

MysteryMind is an AI-powered detective game where players solve dynamically generated mystery cases by collecting clues, interrogating suspects, and identifying the culprit.

## Features
- AI Generated Cases
- Suspect Interrogation
- Evidence Collection
- Interactive Investigation
- Dynamic Story Generation
- Score Tracking

## Tech Stack
Frontend:
- React
- TypeScript
- Tailwind CSS
- Framer Motion

Backend:
- FastAPI
- Python

Database:
- MongoDB

AI:
- Google Gemini API
